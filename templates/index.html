ℹ️ About this Portal

Function:
This is a highly secure, read-only monitoring dashboard designed to provide real-time health metrics, active query analysis, and security validation for SQL Server instances.

How to Use:

  • Log in using your authorized Admin credentials.
  • Click the ➕ (Add) button to securely register new SQL Servers. Passwords are encrypted in SQLite.
  • Select a server from the dropdown and click Refresh Dashboard to view live metrics from the connection pool.
  • Use the specialized tools at the bottom to actively scan for Index Fragmentation (now utilizing Background Caching!) or validate Post-Restore Security.

For your security, the system will automatically lock and log you out after 5 minutes of inactivity.
