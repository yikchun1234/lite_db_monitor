# lite_db_monitor

Step to use
1. Install
   pip install pyodbc
   
2. open python (...location) app.py

3. Start the Backend: Ensure your app.py is running in your terminal. When it runs, it will print out a local web address (usually http://127.0.0.1:5000 or http://0.0.0.0:5000).
  
4. Open the Browser: Open Chrome, Edge, or whatever browser you prefer.
   
5. Enter the Local Server URL: * If you are running the browser on the same machine where Python is running, type: http://localhost:5000 in the address bar.

If you are on a different computer, type the server's IP address: e.g., http://192.168.1.50:5000.
